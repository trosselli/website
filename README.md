## Getting Started

1. Open `index.html` in your web browser
2. Customize the content with your personal information
3. Update the accomplishment cards with your achievements
4. Add your contact information (email, LinkedIn, GitHub)
5. Replace "Your Name" with your actual name throughout the files

## Customization

### Update Personal Information

1. **Name**: Replace "Your Name" in:
   - `index.html` (title, nav-logo, footer)
   - Update meta description if needed

2. **Contact Information**: Update in the contact section:
   - Email address
   - LinkedIn profile URL
   - GitHub username

3. **Accomplishments**: Edit the accomplishment cards with:
   - Your actual achievements
   - Dates
   - Descriptions
   - Icons (emoji or custom icons)

4. **About Section**: Update the about text with your professional summary

### Styling

- Colors can be customized in `styles.css` using CSS variables in the `:root` selector
- Fonts can be changed by modifying the Google Fonts link in `index.html`
